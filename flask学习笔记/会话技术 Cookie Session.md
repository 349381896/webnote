		-支持过期
		-不能跨域名

			·可以直接加载整个网站

	。 session
		-服务端会话技术
		-对数据进行数据安全操作
		-默认在内存中
			·不容易管理
			·容易丢失
			·不能多台电脑协作 
		-安装插件pip install flask-session 默认有效期是31天
